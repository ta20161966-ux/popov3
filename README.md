# Book Tracker

**Book Tracker** — простое приложение для учета и фильтрации книг. Вы можете добавлять книги, сохранять их в файл, загружать из файла и фильтровать по жанру и количеству страниц.

---

## Возможности

- Добавление новой книги (название, автор, жанр, страницы)
- Отображение списка книг
- Фильтрация по жанру и минимальному количеству страниц
- Сохранение данных в JSON файл
- Загрузка данных из JSON файла

---
